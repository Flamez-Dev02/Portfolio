## Overview

This is the professional portfolio of **Daniel Olutoba Iyanda**. Built with a focus on **Premium Aesthetics** and **Technical Precision**, this site showcases advanced front-end capabilities including sophisticated animations, responsive logic-driven layouts, and a modern "Midnight & Cyan" design system.

## Key Features

- **Premium Design System**: A bespoke "Glassmorphism" UI featuring frosted surfaces, neon-glow typography, and a curated midnight color palette.
- **Advanced GSAP Animations**:
  - **Magnetic Interaction**: CTA buttons and icons that react to the user's cursor movements.
  - **Staggered Scroll Reveals**: Fluid section entry animations powered by GSAP ScrollTrigger.
  - **Hero Entrance Sequence**: Professional staggered landing page load sequence for a "Wow" first impression.
- **Live Contact Engine**: Integrated with **Formspree** for real-time email delivery, with client-side form validation.
- **Responsive Architecture**: Pixel-perfect layout optimization for all screen sizes (Mobile, Tablet, Desktop) using Bootstrap 5 and custom media queries.
- **Optimized Performance**: High-efficiency CSS and JavaScript for lightning-fast load times and smooth 60fps animations.

## Technologies Used

- **HTML5**: Semantic structure optimized for SEO and accessibility.
- **CSS3**: Advanced styling including **Glassmorphism**, **Neon Glow**, and custom variable-based design tokens.
- **JavaScript (ES6+)**: Logic-driven interactions, DOM manipulation, and form validation.
- **GSAP (GreenSock)**: The industry standard for high-performance web animations.
- **ScrollTrigger**: For immersive scroll-based storytelling.
- **Bootstrap 5**: Mobile-first responsive grid system.
- **Formspree**: Professional backend-less form handling.
- **Google Fonts (Poppins)**: Premium modern typography.
- **Bootstrap Icons**: Consistent, high-quality iconography.
